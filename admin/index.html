Selamat Datang

Di bawah ini adalah beberapa aturan yang berlaku dalam aplikasi:

  1. Pohon Keputusan:

    Pohon keputusan akan dibangun berdasarkan data CSV yang diinputkan. Pohon ini akan diklasifikasikan dengan aturan dari model yang telah dibentuk menggunakan data latih.

  2. Data Uji:

    Data uji akan digunakan untuk klasifikasi. Proses ini akan melibatkan pencocokan pohon keputusan yang telah dibentuk dengan data uji yang diberikan.

  3. Editing Data:

    Editing data secara langsung akan dilakukan pada file CSV menggunakan Excel. Hal ini bertujuan untuk mempermudah rekapitulasi data dan mempercepat proses pengolahan data.

  4. Proses Otomatis:

    Data dalam file Excel CSV akan diproses secara otomatis saat program dijalankan. Hal ini akan memastikan bahwa data terbaru selalu digunakan dalam proses klasifikasi.
